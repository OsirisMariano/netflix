# Netflix Clone

Clone da interface da Netflix. Migrado de HTML/CSS/jQuery para **React 19 + TypeScript + Vite**.

## Funcionalidades

- **Header** com logotipo e navegação
- **Banner Hero** com filme em destaque
- **Carrossel de filmes** responsivo
- **Design responsivo** (320px a 1920px)
- **Tema escuro** Netflix

## Stack

| Tecnologia | Versão |
|---|---|
| React | ^19.2.7 |
| TypeScript | ~6.0.2 |
| Vite | ^8.1.0 |
| Oxlint | ^1.69.0 |
